# Fashion Shaa POS

A modern, desktop-based Point of Sale application built with Electron.

## 🚀 How to Run
Double-click the **`Launch POS.bat`** file in this folder to start the application.

## 📂 Project Structure
- **`simple-pos/`**: Contains the main source code for the new Electron application.
- **`legacy-java-version/`**: Archives the previous Java/Swing based POS system.
- **`Launch POS.bat`**: Shortcut to launch the application.

## ✨ Features
- **Point of Sale**: Fast calculator-style interface for handling sales.
- **Employee Management**: Manage staff roles and access.
- **Analytics**: View daily, monthly, and yearly sales reports.
- **Printing**: Native system print dialog for receipts.

## 🛠️ Development
To work on the code:
1.  Navigate to `simple-pos`.
2.  Run `npm install` to setup dependencies.
3.  Run `npm start` to launch in development mode.
